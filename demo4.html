Introduction

In any Programming language, there are some fundamentals we need to know before writing programs. This Chapter will also make you to know about such a topic Data Handling which is consists of two parts:   (i) Data Types   (ii) Variables.

C++ provides a predefined set of data types for handling that data it uses. The Data type either be fundamental or derived. The derived data types in C++ are of two types : (i) built-in types, & (ii) user-defined types. Data can be stored in any of these data types. Also, a program needs a means to identify stored data.

Concept of Data Types

Data types are means to identify the type of data and associated operations of handling it.

Data can be of many types e.g. character, integer, real, string etc. Any thing enclosed in single quotes represents character data.Numbers without fractions represents integer data. Numbers with fractions represent real data and anything enclosed in double quotes representing a string.Thatmeans, the data could be dealt with many of types, a programming language must provide ways and facilities to handle all types of data.

In C++ data types are of two types:

     (i) Fundamental types
     (ii) Derived types.

There are total five(5) fundamental data types in C++ programming language : char, int, float, double and void.
char represents "Charcater", int represents "Integer", float represents "floating-point", double represents "double floating-point" and void represents"valueless".

Derived data types are constructed from the above introduced fundamental data types are as:    array, functions, pointers, references, constants, classes, structures, unions and enumerations

C++ Data Types

   As we know, there are five atomic data types in C++; char, int, float, double, and void. All other data types in C++ are based upon one of these atomic types. Such that types are known as derived data types. The size and range of these data types vary with each processor type and withthe implementation of the C++ compiler. Character is of 1 byte, an integer is of 2 bytes, but we cannot make this assumption, if we want our programs to be adaptable to the widest range of computers.
