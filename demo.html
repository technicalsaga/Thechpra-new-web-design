Intoduction

The C++ programming language was developed at AT&T Bell laboratories in early 1980s by Bjarne Stroustrup. He found 'C' lacking for simultaneously and decided to extend the language by adding features from his favourite language Simula 67.

Simula 67 was one of the earliest OOP languages. Bjarne Stroustrup called it "C with classes" originally. The name C++ was coined by Rick Mascitti where"++" is the increment operator. The maturation of C++ language is assested to by two recent events: (1) the formation of ANSI(American National Standard Institute) C++ Committe and, (2) the publication of The Annotated C++ Reference Manual by Ellis and Stroustrup.

The major reason behind the success and popularity of C++is that i supports the object-oriented technology

One can easily judge the importance of C++ by reading this : "Object-oriented Technology is regarded as the ultimate paradigm for the modelling of information, be that logic or data. The C++ has to fulfill this goal"

C++ Character Set

Character set is set of valid characters that a language can recognise. A character represents any letter, digit, or any other sign. The C++ has the following character set

Letters A-Z, a-z

Digits 0-9

Special Symbols Space + - * / ^ \ () [] {} = != < > . '" $ , ; : % ! & ? _(underscore) # <= >= @

White Spaces Blank Space, Horizontal tab(--->), carriage return(<_|), Newline, Form feed.

Other Characters C++ can process any of the 256 ASCII characters as data or as literals.

Tokens:-

The smallest individual unit in a program is known as a token.

C++ has the following tokens:

1. Keywords

2. Identifiers

3. Literals

4. Puncuators

5. Operators

Procedural

Procedural programming is a programming paradigm, derived from a structured programming, based on the concept of the procedure call.

OOP

Object-oriented programming paradigm based on the concept of "objects" which can contain data, in the form of fields, and code, in the form of procedures.

Objects

Object is an identifiable entity with some characterstics and behaviour.

Class

A Class is a group of Objects that shares common properties and Relationships.
