Tokens:-

The smallest individual unit in a program is known as a token.

C++ has the following tokens:
1. Keywords
2. Identifiers
3. Literals
4. Puncuators
5. Operators

1. Keywords

Keywords are the words that convey a special meaning to the language complier. These are reserved for special purpose and must not be used as normal identifier names.

The original C++(developed by Stroustrup) contains the following keywords:

asm    auto    break    case    catch    char    class    const    continue    default    delete    do    double    else    enum    extern    float    for    friend    goto    if    inline    integer    long    new    operator    private    protected    public    register    return    short    signed    sizeof    static    struct    switch    template    this    throw    try    typedef    union    unsigned    virtual    void    volatile    while    

In addition, identifiers containing a double underscore(_ _) are reserved for use by C++ implementations and standard libraries and should be avoided by users.

2. Identifiers

An identifier is an arbitrarily long sequence of letters and digits.

   Rules for Assigning Identifiers:
    * It contains: (A-Z, a-z, 0-9, _).
    * All characters are significant.
    * Identifiers are fundamental building blocks of a program.
    * First character must be letter; the underscore(_) also considered as letter.
    * Do not contain any special character other than (_).

3. Literals

Literals are data items that never change their value during a program run.
    It is also reffered to as constants.

   Kinds of Literals:
(i) integer-constant;   (ii) character-constant;   (iii) floating-constant;   (iv) string-constant;   

(i) Integer-constant:- Integer constant are whole numbers without any fractional part.
    * Must contain at least one digit.
    * Do not contain any decimal point.
    * It may contain either(+)or(-) sign.
    * Commas cannot appear in an integer constant.

C++ allows three types of integers constant:
(a) Decimal(base10);  (b) Octal(base 8);  (c) Hexadecimal(base 16);  

(a)Decimal Integer Constant: An integer constant consisting of a sequence of digits is taken to be decimal integer constant unless it begins with 0. eg:- 1234,41, +73, -13 are decimal integer constant.
(b)Octal Integer Constant: A sequence of digits starting with 0 is taken to be octal integer. eg:- 8 will be written as 010 as octal integer(810= 108)
(c)Hexadecimal Integer Constant: A sequence of digit precceded by 0x or 0X is taken to be an hexadecimal integer. eg:- 12 will be written as 0XC(as hexadecimal).
(ii) Character-constant:- A character constant is single character enclosed in single quotes, as in 'a'.
* Non-graphic characters can be represented by using escape sequence.

Escape Sequence Non-graphic Character
\a Audible bell(alert)
\b Backspace
\f Formfeed
\n Newline or linefeed
\r Carriage Return
\t Horizontal tab
\v Vertical tab
\\ Backslash
\' Single quote
\" Double quote
\? Question mark
\On Octal number (On reoresents the number in octal)
\xHn Hexadecimal number (Hn represents the number in hexadecimal)
\0 Null
An escape sequence represents a single character

(iii) Floating Constant:-It is also known as eal constants.

Real constants are numbers having fractional part. These may be written in one of the two forms called fractional form or the exponent form.

   *A real Constant in fraction form consists of signed or unsigned digits including decimal point between digits. eg:- 2.0, 17.5, -13.0, -0.00654;

   *A real constant in exponent form consist of two part: mantissa and exponent.
eg:- 6.8 is written as 0.68* 101 = 0.68 E01, where mantissa part is 0.68 and the exponent part is 1.
E01 represents 101 eg:- 152E05, 1.52E07, etc...

(iv) String Literals:-Multiple character constants are dealt with two ways in C++.

If enclosed in single quotes, these are treated as character constants and if enclosed in double quotes, these are treated as string literals

"A string literals is a sequence of characters surrounded by double quotes."

Each string literal is added with a '\0'(null) terminating character.

Thr string "abc" actually represented as "abc\0" in the memory. It's size is 4 character (including of terminator character.).

4. Punctuators

The following characters are used as punctuators (also known as separators) in C++:

[] () {} , : ; * ... = #

that enhances a program's readiability.

5. Operators

Tokens that trigger some computation or action when applied to variables and other objects is an expression.

Input/Output (i/o) operators (<< and >>), arithematic operators (+, -, *, /, %), increment/decrement operators (++, --), relational operators (==, !=, <,>,<=,>=), logical operators (&&, ||, !) and conditional operators (?:).
