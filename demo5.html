Fundamental Data Types

Fundamental(Atomic) data types are those that are not composed of other Data Types.

int data type "Data type for integers" Integers are the whole numbers which doesn't have any fractional part. In C++ Integers are represented by int data type.
Any Identifier when declared in as int bears only Integers values.Integers can be negative(-) or positive(+) but whether integers can have fractional values, it depends upon the integer types.

char data types "Data type for Character" Characters can store any member of the C++ implementation's basic character set. An Identifier declared as char becomes a character variable.

    char type is often said to be an integer type. It is said because letters, symbols etc are represented in memory by associated number codes.It is very easy to code numbers into binary. Thus, even letters and symbols are also associated numbers codes(eg:- ASCII code), because of which data type char is called another integer type.

  Characters   ASCII Values
  '0'-'9'        48-57
  'A'-'Z'        65-90
  'a'-'z'        97-122
 other characters        0-255 excluding the above mentioned codes

float Data Type "Data Type for floating-point numbers" Any number having fractional part is a floating-point number. eg:- 45.76 is a floating -point number.
The decimal point signals that it is a floating-point number not an integer. The number 23 is an integer but 23.0 is a floating-point number.Floating-point numbers can also be written in exponent notation. eg:- 14.567 would be written as 1.4567E01. An Identifier declared asfloat becomes a floating-point variable and can hold floating-point numbers.Floating point numbers Represents real numbers, which are used for measurable quantities and typically6 have a fractional part.

          Floating-point numbers have two advantages over integers. First, they can represent values between the integers. Second, they can represent a much greater range of values.

double Data Type "Data Type for double precision floating-point numbers" This Data Type is also used for handling floating-point numbers. But it is treated as a distinct data type because, it occupies twice as much memory as type float, and stores floating point numbers with larger range and precision.
  The type double is larger and slower than type float.

void Data Type "Data Type for empty set of values and non-returning functions" The void specifies an empty set of values. It is used as the return type for functions that do not return a value. No object of type void may be discussed.


Data Type Modifiers

Except data type image void, the basic data types may have various modifiers preceding them. We use a modifier to alter the meaning of the base type to fit various situation more precisely. The Modifiers are as below:
        signed
        unsigned
        long
        short

Note: Here, ANSI standards eliminates long float because it is nothing but double.

Memory of Computer is organised into bytes. A bytes consists of 8-bits, where a bit is the smallest unit of memory. 1 bit represents 2 unique combinations of binary numbers 0 and 1.
That is a byte 8 bits can represents 28=256 unique number of combinations. 2-bytes represents 65536 unique number of combinations.

Integer Types

short is short name for short int and long is short name for long int.

type Approximate Size(in bytes) Minimal Range
short 2 -32,768 to 32,767
unsigned short 2 0 to 65,535
signed short 2 same as short
int 2 -32,768 to 32,767
unsigned int 2 0 to 65,535
signed int 2 same as int
long 4 -2,147,483,648 to 2,147,483,647
Unsigned long 4 0 to 4,294,967,295
signed long 4 same as long

              short, int and long are signed by default

Character Types

The char type is another integer type because inside memory it actually holds numbers eqivalent codes of character/symbols.

The char type can also be signed or unsigned and unlike int, char is neither sgned or unsigned by default.

type Approximate Size(in bytes) Minimal Range
char 1 -128 to 127
unsigned char 1 0 to 255
signed char 1 same as char
